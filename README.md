# Ranks Plugin

[![Project Status: Completed](./assets/images/badges/status.svg)](./)

> A simple rank system project.

## Features

- Saves ranks to a `.yml` file
- Supports prefixes
- Supports permissions
- Tab list players are sorted based on their rank
